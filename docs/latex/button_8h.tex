\doxysubsection{Detailed Description}
Interface de gestion des boutons dans Tap\+Venture. 

Ce fichier définit les structures et fonctions nécessaires pour créer, afficher, et gérer les boutons dans l\textquotesingle{}interface utilisateur. Il inclut également des outils pour manipuler les textures, les événements de clic, et les états des boutons. \begin{DoxyAuthor}{Author}
Noam MANFALOTI 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
2 avril 2025 
\end{DoxyDate}
