\doxysubsection{Detailed Description}
Interface de gestion de l\textquotesingle{}affichage des éléments UI dans Tap\+Venture. 

Ce fichier définit les structures et fonctions nécessaires pour gérer l\textquotesingle{}affichage des textes et des images dans l\textquotesingle{}interface utilisateur, ainsi que leur mise à jour. Il inclut également des outils pour manipuler les dimensions relatives à l\textquotesingle{}écran. \begin{DoxyAuthor}{Author}
Noam MANFALOTI
