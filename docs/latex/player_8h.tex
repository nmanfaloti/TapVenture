\doxysubsection{Detailed Description}
Déclarations pour la gestion du joueur dans Tap\+Venture. 

Ce fichier contient les définitions, déclarations et macros nécessaires pour la gestion des données du joueur, incluant le suivi de l\textquotesingle{}or et des dégâts infligés lors des interactions de jeu.
