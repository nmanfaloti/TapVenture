\doxysubsection{Detailed Description}
Structure représentant une notification dans l\textquotesingle{}interface utilisateur. 

Cette structure contient les informations nécessaires pour afficher une notification, telles que le texte, l\textquotesingle{}image de fond, la durée et les positions.
