\doxysubsection{Detailed Description}
Déclarations pour la sauvegarde et le chargement des données de Tap\+Venture. 

Ce fichier contient les déclarations des fonctions permettant de lire, écrire et modifier les sauvegardes au format JSON pour le jeu Tap\+Venture, incluant la gestion des sauvegardes du joueur, des héros et du prestige.
