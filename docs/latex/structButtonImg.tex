\doxysubsection{Detailed Description}
Structure représentant un bouton avec une image dans l\textquotesingle{}interface utilisateur. 

Cette structure contient les informations nécessaires pour afficher et gérer un bouton avec une image, y compris ses dimensions, ses textures, et les fonctions associées.
