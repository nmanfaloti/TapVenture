\doxysubsection{Detailed Description}
Interface de gestion audio pour Tap\+Venture. 

Ce fichier définit les types et fonctions permettant la gestion de l\textquotesingle{}audio dans Tap\+Venture, incluant le chargement, la lecture et la libération des ressources audio pour les musiques et effets sonores. \begin{DoxyAuthor}{Author}
Lucas DUPONT
