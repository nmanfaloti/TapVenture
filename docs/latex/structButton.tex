\doxysubsection{Detailed Description}
Structure représentant un bouton dans l\textquotesingle{}interface utilisateur. 

Cette structure contient les informations nécessaires pour afficher et gérer un bouton, y compris ses dimensions, ses textures, son texte, et les fonctions associées.
