Detailed Description

Interface de gestion des chaînes de caractères dans TapVenture.

Ce fichier définit les constantes, types et fonctions pour l'initialisation et la gestion des chaînes de caractères utilisées dans TapVenture, incluant la concaténation, le formatage et la conversion de nombres.
