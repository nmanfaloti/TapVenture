Detailed Description

Interface de gestion de l'affichage des éléments UI dans TapVenture.

Ce fichier définit les structures et fonctions nécessaires pour gérer l'affichage des textes et des images dans l'interface utilisateur, ainsi que leur mise à jour. Il inclut également des outils pour manipuler les dimensions relatives à l'écran.
