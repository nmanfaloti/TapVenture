Detailed Description

Interface de gestion des boutons dans TapVenture.

Ce fichier définit les structures et fonctions nécessaires pour créer, afficher, et gérer les boutons dans l'interface utilisateur. Il inclut également des outils pour manipuler les textures, les événements de clic, et les états des boutons.
