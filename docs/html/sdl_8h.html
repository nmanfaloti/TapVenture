Detailed Description

Déclarations pour l'initialisation, la gestion et la fermeture du sous-système SDL dans TapVenture.

Ce fichier contient les déclarations des fonctions d'initialisation de SDL, de gestion de la fenêtre et du renderer, ainsi que celles permettant de libérer les ressources associées (textures, polices et curseurs).

Function Documentation

◆ init_SDL()

int init_SDL ( )

Initialise le sous-système SDL et ses modules associés.

Cette fonction initialise SDL, SDL_ttf, SDL_image et SDL_mixer. Elle crée également la fenêtre et le renderer utilisés pour afficher le jeu.

Returns
0 en cas de succès, un code d'erreur sinon.

◆ SDLExit()

void SDLExit ( )

Ferme SDL et libère toutes les ressources associées.

Cette fonction détruit la fenêtre, le renderer, les curseurs, libère les polices et appelle les fonctions de fermeture pour SDL_mixer, SDL_ttf et SDL_image.

◆ SDLScreenHandler()

void SDLScreenHandler ( )

Gère le rafraîchissement de l'écran.

Cette fonction efface le renderer, met à jour la scène et affiche le contenu rendu à l'écran. Elle est généralement appelée dans la boucle principale du jeu.
