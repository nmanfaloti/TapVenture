Detailed Description

Structure représentant un challenge dans le jeu.

Contient les informations essentielles pour gérer un challenge, notamment son état, le temps de démarrage, la durée, la cible à atteindre, la récompense octroyée, le cooldown ainsi que le timestamp de la dernière exécution.
